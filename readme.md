# Task app
## Start
```bash
#Install modules
yarn install
#Start in dev mode
yarn start
#Open http://localhost:8080/
```
## Build
```bash
#Install modules
yarn install
#Start build
yarn build
```
## Build - production mode
```bash
#Install modules
yarn install
#Start build in production mode
yarn build-prod
```
## Lint project
```bash
#Install modules
yarn install
#Start in dev mode
yarn lint
```
